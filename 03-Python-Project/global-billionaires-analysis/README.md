# Global Billionaires Analysis  
### Python Data Analysis Project  

---

## Project Overview  

This project analyzes global billionaire data using Python to uncover insights about wealth distribution, geographic concentration, and industry dominance.

The analysis focuses on:

- Total billionaire count
- Top countries by billionaire population
- Industry distribution
- Wealth statistics (average, total, highest net worth)
- Country-level wealth comparison

This project demonstrates practical data analysis, visualization, and reporting using Python.

---

## Dataset Overview  

- Total Billionaires Analyzed: **475**
- Analysis Date: **February 2026**
- Data Format: Excel (.xlsx)

### Key Columns Used:
- `personName`
- `country`
- `industries`
- `finalWorth`

---

## Key Insights  

### Top Countries by Billionaire Count
- United States (190)
- China (73)
- United Kingdom (22)
- India (20)
- Germany (20)

The United States dominates global billionaire concentration.
