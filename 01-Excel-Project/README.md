# Excel Project: University of Ibadan Enrollment Analysis

## Overview
Analyzed 8,673 student enrollment records from 2021-2024 across 17 faculties.

## Skills Demonstrated
- Data cleaning (8,600+ records)
- Pivot tables
- Dashboard creation
- Data visualization

## Key Findings
- 4% enrollment decline over 3 years
- Arts faculty leads with 1,816 students
- Gender distribution: 52% male, 48% female
