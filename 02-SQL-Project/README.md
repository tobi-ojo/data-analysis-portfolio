# SQL Project: Database Query Analysis

## Overview
Demonstrated SQL proficiency through querying and analyzing datasets using MySQL Workbench.

## Skills Demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY aggregations  
- JOIN operations
- CASE statements
- Subqueries
- REGEXP pattern matching
- IF functions

## Tools Used
- MySQL Workbench
- Database: richlist
- Tables: billionaires, geography

## Query Examples
This project showcases 8+ SQL queries demonstrating various techniques:
- Data filtering and sorting
- Aggregations and counts
- Multi-table joins
- Conditional logic
- Pattern matching

## Key Learnings
Through this project, I learned to extract insights from databases, combine data from multiple sources, and write efficient queries for data analysis.
